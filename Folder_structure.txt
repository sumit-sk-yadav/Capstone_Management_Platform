### Key Notes

1. **Environment Files**: `.env` and `.env.local` should NEVER be committed to Git
2. **Migrations**: Each Django app should have a `migrations/` folder (created with `makemigrations`)
3. **Static Files**: Django will collect static files into `backend/static/` with `collectstatic`
4. **Media Files**: User uploads go to `backend/media/`
5. **TypeScript**: All frontend files use `.tsx` for React components and `.ts` for utilities
6. **App Router**: Frontend uses Next.js 14+ app router (not pages router)
